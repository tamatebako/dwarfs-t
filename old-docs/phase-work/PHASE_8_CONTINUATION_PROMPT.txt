Pattern 2 - Job Wrapper Class (OCP):
```cpp
template<typename Func, typename... Args>
class job_wrapper {
    Func func_;
    std::tuple<Args...> args_;
public:
    void operator()() { 
        std::apply(func_, std::move(args_)); 
    }
};
```
